# FFmpeg Directory

Place your FFmpeg binaries here for bundled distribution.

## Required Files

For Windows:
- `ffmpeg.exe`
- `ffprobe.exe`

For Linux:
- `ffmpeg`
- `ffprobe`

## Download FFmpeg

- **Windows**: https://www.gyan.dev/ffmpeg/builds/
- **Linux**: Use `apt install ffmpeg` or download from https://ffmpeg.org/
- **macOS**: Use `brew install ffmpeg`

## Note

FFmpeg binaries are not included in this repository due to size.
The application will use system FFmpeg if available, or you can place binaries here.
